Project Overview:
PervasiveAeroAgents develops intelligent multi-drone systems for post-disaster response by combining AI, multimodal sensing, secure communication, and proof-of-concept deployment. The project supports search and rescue through coordinated UAV swarms capable of victim detection, autonomous navigation, and resilient operation in complex environments.

Main Objectives:
- Develop a multi-drone swarm system for coordinated post-disaster operations
- Define a multi-tier drone architecture
- Integrate sensing technologies for human detection under rubble
- Maximize mission lifetime through coordinated wireless charging
- Design AI-based techniques for object detection, obstacle avoidance, and autonomous navigation
- Develop secure group communication
- Build a proof of concept with a cyber-physical digital twin, hardware, and software deployment

Work Packages:
WP1: Multi-drone Architecture and System Design
Focus: architecture, coordination, task allocation, path planning, wireless charging, adaptive redeployment

WP2: AI Detection and Autonomous Navigation
Focus: human detection under rubble, multimodal sensing, reinforcement learning, online learning, autonomous navigation

WP3: Secure Communication
Focus: mobility-aware secure communication, robust group protocols, resilient UAV coordination, security-overhead tradeoff

WP4: Digital Twin and Prototype
Focus: drone assembly, sensor integration, Gazebo/ROS2 digital twin, prototype development, field testing
